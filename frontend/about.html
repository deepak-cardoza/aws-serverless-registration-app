About the Application

Learn more about the architecture, services, and technologies used

What it does

This application is a fully functional user authentication system designed as a learning resource and demonstration of modern serverless cloud architecture. Users can securely register new accounts and log in to view their private profiles. The frontend communicates with a set of decoupled backend REST APIs to securely validate credentials and persist data.

AWS Serverless Infrastructure

The backend is entirely serverless, ensuring high scalability, zero server management overhead, and a pay-as-you-go cost structure.

AWS Lambda

Handles business logic (e.g., registration processing and login validation) as discrete, event-driven serverless functions.

Amazon API Gateway

Acts as the front door for backend services, providing secure, CORS-enabled HTTP endpoints (/register and /login) routed to Lambda.

Amazon DynamoDB

A fast, flexible NoSQL database service used to store user credentials and profile details reliably at any scale.

AWS SSM Parameter Store

Stores centralized configuration parameters (such as the Lambda Layer ARN) used across various deployment services.

Amazon CloudFront

A global content delivery network (CDN) that distributes the static frontend assets securely and at low latency.

Amazon S3

Hosts the static HTML, CSS, and client-side JavaScript files, accessed securely by CloudFront using Origin Access Control (OAC).

Technology Stack

HTML5, CSS3 & JavaScript (ES6+)

Powers the user interface with a modern, responsive layout, smooth transition animations, and direct API integrations using Axios.

TypeScript & Node.js

Used to write type-safe, clean backend handlers compiled to standard JavaScript running on Node.js runtimes.

Serverless Framework v3

Utilized as the Infrastructure as Code (IaC) tool to configure, provision, and deploy all cloud resources programmatically.

Zod Validation

Performs strict, run-time schema validation for registration payloads to ensure data consistency and system stability.

bcryptjs & JWT

Secures user passwords using salt-rounded hashing and authenticates user requests using JSON Web Tokens (JWT).
